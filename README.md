# 🧠 Neuroleadership – SCARF Modell

## 📌 Überblick
Dieses Projekt analysiert das SCARF‑Modell (Status, Certainty, Autonomy, Relatedness, Fairness) und seine Bedeutung für Führung und Change‑Management.

---

## 🎯 Ziele
- Analyse des SCARF‑Modells  
- Bewertung organisationaler Auswirkungen  
- Ableitung von Führungsoptionen  

---

## 📁 Projektstruktur
docs/
visuals/
src/

---

## 🚀 Ergebnisse
Das SCARF‑Modell bietet klare Handlungsempfehlungen für Führungskräfte in Veränderungsprozessen.
